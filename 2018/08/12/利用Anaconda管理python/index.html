利用Anaconda管理python

Posted on 2018-08-12 | In 编码

目的

为了便于在Linux环境中对Python多版本，多环境管理。Anaconda是强有力的工具。本文在Ubuntu 16.04中安装Anaconda 4.3.0（python 3.6）。

下载

由于官网下载速度较慢，所以选择清华镜像进行下载。具体链接https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/。下载所需版本，本文下载Anaconda 4.3.0

安装

  • 利用bash命令进行安装，按下回车键盘继续安装
  • 以下为协议，回车至显示完，键入yes同意，出现如下选项
  • 此处键入安装路径后(注意安装的最终路径必须不存在，会自动新建目录))，开始安装，安装完之后，提示写入环境变量，键入yes，则安装完成。
  • 重启终端，键入anaconda -v查看版本，键入conda list查看已安装模块

管理虚拟环境

  • 创建

    键入如下命令创建虚拟环境，环境名为name输入值

    conda create -n [name]

  • 激活

    source activate [name]

  • 退出环境

    source deactivate [name]

  • 删除环境

    conda remove -n [name] --all

  • 查看已经安装环境列表

conda info -e
